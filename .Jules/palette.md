# 🎨 Palette's Journal - Harpia Consultoria

This journal records critical UX and accessibility learnings from the development of the Harpia Consultoria landing page.

## 2025-05-15 - Improving Keyboard Accessibility
**Learning:** Large static landing pages often lack basic accessibility features like "Skip to Content" links, which hinders navigation for keyboard and screen reader users. Additionally, relying solely on default browser focus indicators can be insufficient on dark backgrounds.
**Action:** Always implement a skip-to-content link as the first interactive element and define custom, high-contrast focus-visible styles.
