## 2025-05-22 - Accessibility via Tailwind Utilities
**Learning:** For projects using Tailwind CSS, implementing accessibility features like skip links should prioritize standard utility classes (e.g., `sr-only`, `focus:not-sr-only`) over custom CSS to maintain consistency and adhere to project constraints.
**Action:** Always check `tailwind.config.js` for available tokens and use utility classes for micro-UX improvements.
