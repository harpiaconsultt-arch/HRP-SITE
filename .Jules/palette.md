## 2025-05-15 - [Skip to Content Link Implementation]
**Learning:** Implementing a 'Skip to Content' link is a critical micro-UX improvement for keyboard accessibility, especially on pages with large navigation menus. Using Tailwind utility classes like `sr-only focus:not-sr-only` is an efficient way to ensure the link is only visible when focused.
**Action:** Always check for a 'Skip to Content' link as the first interactive element in any web project to ensure WCAG compliance and improved keyboard navigation.
